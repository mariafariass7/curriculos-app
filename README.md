# Sistema de Gestão de Currículos

Aplicação web desenvolvida com Next.js para gerenciamento de currículos, focada em organização, validação de dados e experiência do usuário.

---

## 🚀 Tecnologias utilizadas

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS
- React Hook Form
- Yup (validação)
- Sonner (toasts)
- React Icons

---

## 📌 Funcionalidades

- Cadastro de currículos
- Listagem de candidatos
- Página de detalhes dinâmica
- Formulário com validação completa
- Campos dinâmicos (experiência e formação)
- Persistência de dados com localStorage
- Máscaras de CPF e telefone
- Feedback visual com notificações

---

## 🧠 Estrutura do sistema

- `/` → Página inicial
- `/sistema/paginas/curriculos` → Lista de currículos
- `/sistema/paginas/curriculos/[id]` → Detalhes do currículo
- `/sistema/paginas/curriculos/novo` → Cadastro
